# What is Linux?
**Linux**is the **kernel base operating system**

👉 It acts as a bridge between:
 - **User (you)**
 - **Hardware (CPU, memory, disk, etc.)**

🔹 Simple Meaning:

Linux helps you:

 - Run programs
 - Manage files
 - Control hardware
