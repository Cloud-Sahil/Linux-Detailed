1. **host name** -   To check machine name
2. **pwd** -   Present Working directly
3. **$** -   Local User
4. **lscpu** -   To check cpu
5. **lscpu|less** - scroll
6. **Press Q** - For Quit
7. **Free** - To check memory
  - **free -k** - kb
  - **free -m** - mb
  - **free -g** - gb
  - **free -h** - iga byte
