# User Management
**User management** means creating, modifying, deleting users and controlling their access/permissions in a Linux system.

---
## User Types
1. **local user** --------> limited access, /home, /bin, $ , range 1000-65000
2. **super user** -------> admin access  , /root, /sbin or bin,#, 0-499 ,root user - 0 (id)
3. **system/service user** -------->  perform specific service or system related task
