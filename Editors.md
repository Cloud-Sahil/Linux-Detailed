# Editors (Linux) 

 - **1. nano**
 - **2. pica**
 - **3. vi**
 - **4. vim**

---
## 1. Nano Editor
```sh
nano file.txt
```
### Example:
 - Write text directly
 - Press **Ctrl + O** → Save
 - Press **Ctrl + X** → Exit
---
## 2. Pico Editor
```sh
pico file.txt
```
### Example:
 - Edit file easily
 - Save → **Ctrl + O**
 - Exit → **Ctrl + X**
---
## vi Editor
```sh
vi file.txt
```
### Example:
 - i ------------------> Go to Insert Mode
 - Esc ------------------> Go out of insert mode / go to command mode
 - :w ------------------> To save the changes
 - :wq ------------------> To save & exit
 - :q! ------------------> To exit without save
## vim Editor
```sh
vim file.txt
```
### Example:
 - i ------------------> Go to Insert Mode
 - Esc ------------------> Go out of insert mode / go to command mode
 - :w ------------------> To save the changes
 - :wq ------------------> To save & exit
 - :q! ------------------> To exit without save
