# Permission Management
**Permission management** controls who can **read, write, or execute** files and directories.

1. r ----------> read (4)  
2. w -----------> write (2)      
3. x -----------> execute (1) 
4. (-) -----------> no permission (0)
 - total sum is 7

1. chmod ---------- to change permission
2. chown ----------- to change owner
3. chgrp ------------ to change group

1. u ----------> user / owner		       	Add (+)
2. g ----------> group			              remove (-)
3. o ---------> other		               	set (=)
